# 📚 PC_Knowledges_Library

> 一个面向初学者的计算机知识库，帮你从零开始认识计算机的世界。

## 📖 内容概览

```
📂 购买记/          — 选购计算机的实用指南
   ├── 前述.md      — 起笔前的思想准备
   ├── 认识计算机类型.md  — 台式机、笔记本、一体机怎么选
   └── 认识计算机硬件.md  — CPU、GPU、主板……各部件详解

📂 使用记/          — 计算机使用小技巧
   └── 常见的键盘快捷键.md — 提升效率的快捷键合集

📂 引用图片/        — 文章中使用的插图资源
   └── 图片使用申明.md

📄 知识库前言.md    — 知识库的缘起与初衷
📄 开发者：语法.md  — 本知识库的 Markdown 写作规范
```

## 🎯 目标读者

- 对计算机感兴趣的**零基础初学者**
- 想要**选购电脑**但又不太懂硬件的小白
- 希望掌握**实用计算机技能**的普通用户

## ✨ 写作特色

- **"煮包"式风格** — 亲切易懂，像朋友边煮茶边给你讲电脑知识
- **Callout 提示框** — 重要知识点、注意事项一目了然
- **图文并茂** — 配有硬件图片，直观理解
- **互动测验** — 每篇文章末尾有自测题，巩固知识

## 🚀 同步机制

本仓库通过 **GitHub Actions** 每周自动从 OneDrive 同步最新内容。
