# 🧠 暝烁的知识库

主子（孙梓豪）的个人计算机知识库，记录从购买到使用、维护、维修的全流程学习笔记。

> 🌐 **GitHub Pages 站点**：[https://zihao-0227.github.io/PC_Knowledge_Library/](https://zihao-0227.github.io/PC_Knowledge_Library/)

## 知识库结构

| 目录 | 说明 |
|------|------|
| `购买记/` | 配置、验机与配电脑 |
| `使用记/` | Windows系统工具、快捷键 |
| `上网记/` | 浏览器、搜索引擎与网络安全 |
| `软件记/` | 常用软件与专业软件的下载、安装与使用 |
| `维护记/` | 重装系统以及系统封装 |
| `维修记/` | 送修前的简单诊断 |
| `引用图片/` | 知识库中的外部图片资源 |

## 同步机制

本仓库通过 GitHub Actions 自动从 OneDrive 同步内容，每周六 08:00（北京时间）自动更新。
