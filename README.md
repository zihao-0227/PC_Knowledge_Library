# 🧠 PC_Knowledge_Library

你好，欢迎来到这里。

智能手机时代，许多计算机上繁琐的操作已经被简化了。但，计算机依然是性能要求极高工作的得力工具；计算机依旧是许多大学生必不可少的工具之一。可是，由于后代们是在智能手机时代的环境下长大的，这就导致一个极为尴尬的现象：***年轻一代已经越来越不懂得如何使用电脑了***。

这个知识库，我将一个准大学生在从购买到使用再到维护与维修的顺序讲解与解答如何从0入门电脑。
