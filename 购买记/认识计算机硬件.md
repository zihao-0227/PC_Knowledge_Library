# CPU（Central Processing Unit，中央处理器）

它应该长这样👇：

![[CPU.jpeg]]

除了主板，第二重要的硬件就是CPU了。CPU的日常工作就是处理各种各样复杂的数据，所以人们也常称CPU是一台计算机的“电子大脑”。介于CPU是计算机的性能的决定性因素之一，下文中煮包会简单介绍以下CPU选购指南。

## CPU选购

目前市面上主流CPU厂商无非就两个：英特尔（Intel）和AMD（Advanced Micro Devices, Inc.，超威半导体）。其他国产CPU品牌这里就不介绍选购了。

### 英特尔

我们以三个CPU为例：i5-12400F、 i7-1195G7、Ultra 5 245KF，来拆解每一部分的含意。

首先是以i或Ultra开头的部分，这就是英特尔[^1]对于产品线和产品本身的定义。以i开头的是英特尔的酷睿系列，以Ultra开头的是英特尔的酷睿Ultra系列。

> [!QUESTION] 酷睿系列和酷睿Ultra系列有什么不同
>酷睿Ultra系列与酷睿系列主要有以下三个区别：
>
>- 酷睿Ultra系列比酷睿系列的本地AI能力更强。
>- 酷睿Ultra系列比酷睿系列的核显能力更强。
>- 酷睿Ultra系列的笔记本比酷睿系列的笔记本更省电。
>
>总结起来就是：**如果追求性价比，就选择酷睿系列。如果对AI有刚需，或者想体验新CPU的特性，才选择酷睿Ultra系列。**

紧跟i或Ultra后的一般是3、5、7、9这四个数字，一般来讲，数字越大，性能越强。以酷睿系列为例（酷睿Ultra系列参照下文酷睿系列）：

- **i3**：入门
- **i5**：主流
- **i7**：高端
- **i9**：旗舰

> [!WARNING] i9或Ultra 9性能就是最强吗？
> 不一定。一般要参考横线后的代数，同一代数对比下i9或Ultra 9才是同代中最强的。如果跨代对比，这一方法就失效了。电脑城的奸商常常就使用这一信息差来哄骗小白

接下来就是横线的后面的代码：酷睿系列横线的两位或一位（具体情况需要参考长短）是指CPU的代数，如：i5-12400F就是第12代酷睿系列产品，而i7-7700就是第7代酷睿系列产品。

> [!TIP] 试一试
> i7-1195G7是第几代英特尔酷睿产品？
> 
> 答：i7-1195G7是第11代英特尔酷睿产品。

接下来就是后面的数字和字母了。从代表代数的一位或者两位数字过后一直到第一个字母前的所有数字就是这款产品的SKU（Stock Keeping Unit，库存量单位）一般来讲，在同代中，SKU越大，CPU性能越强。如i5-12400F的SKU就是400，与其对比的i3-12100F的SKU就是100。对小白来讲，一般来说，同代中SKU越大，CPU性能越强。所以这两颗CPU中，i5-12400F性能更强。

最后就是最后面的字母了，由于字母有些复杂，如果有需要可以参考下表：

| 常见台式电脑CPU后缀 | 含义                               |
| ----------- | -------------------------------- |
| F           | 无核显，需要额外配置独立显卡                   |
| K           | 不锁倍频，可超频（CPU超载运行，仅限专业人士，不建议小白尝试） |
| KF          | 无核显+可超频（专业人士的台式机首选）              |
| T           | 低功耗，CPU更省电                       |
| X           | 至尊发烧平台，需要搭配定制主板使用（不建议）           |
| KS          | 官方特挑超频版，同代中性能最强（家里有矿的可以上）        |

| 常见笔记本电脑CPU后缀        | 含义                               |
| ------------------- | -------------------------------- |
| Y                   | 功耗极低（7W以下），性能极弱（二合一平板、超级轻薄本常用）   |
| U                   | 功耗较低（15W），性能较弱（轻薄本常用）            |
| H                   | 功耗较高（45W），性能较强（游戏本、全能本常用）        |
| HX                  | 功耗极高（55W+），性能极强（旗舰游戏本，移动工作站常用）   |
| G1/G4/G7（酷睿11代以后弃用） | 带独立核显，G后面的数字代表核显性能等级，数字越大，核显性能越强 |

### AMD

AMD的CPU产品主要就锐龙（Ryzen R[^2]）系列了，其命名规则与英特尔的CPU命名规则基本一致，如R7-7800X3D，这颗CPU就被AMD定位为：锐龙第七代高端CPU；其SKU为800；具有3D 堆叠缓存技术。

下表为AMD常见的后缀名：

| 常见台式电脑CPU后缀 | 含义                                |
| ----------- | --------------------------------- |
| X           | 高频版，可超频                           |
| X3D         | 具有3D堆叠缓存（3D V-Cache）技术，打大型游戏会很有帮助 |
| G           | 带有高性能核显                           |
| GE          | 带有低功耗核显                           |
| XT（现已弃用）    | 官方超频版                             |
| WX          | 性能强悍，工作站专用                        |

| 常见笔记本电脑CPU后缀 | 含义                                    |
| ------------ | ------------------------------------- |
| HS           | 轻薄高性能（35W）                            |
| C            | Chromebook（国外爆火的上网本，国内不常见，一般要求极低功耗）专用 |
| Ue           | 超低功耗增强版                               |

# GPU（Graphics Processing Unit，图形处理器，简称显卡）

它应该长这样👇：

![[GPU.jpg]]

GPU，也就是我们常说的显卡，是负责处理图像和视频信号的硬件。它的主要工作就是把CPU处理好的数据转化为我们能在显示器上看到的画面。

> [!QUESTION] 什么是核显？什么又是独显？
> 核显指部分集成在CPU里的显卡，这种显卡本质上就是为了在没有独立显卡下，计算机处理过的画面至少可以显示在屏幕上，而不至于黑屏（“亮屏卡”）。一般性能较弱。适合平时办公刷剧
> 
> 而独显就是独立于CPU之外的显卡，这种显卡一般来讲性能强悍，适合视频剪辑师、3D建模师、AI绘画开发者这些有特殊需求的工作者和追求极致画质的3A游戏玩家。

显卡的优劣同时也是决定电脑性能的核心因素之一。那么接下来，煮包将会介绍显卡选购指南。
## 显卡选购指南

目前的独立显卡市场主要有两家公司——英伟达（NVIDIA）和AMD（Advanced Micro Devices, Inc.，超微半导体），也就是俗称的N卡和A卡。

> [!INFO] 关于英特尔显卡
> 英特尔近年来也推出了Arc系列独立显卡。不过嘛，英特尔的主要业务还是CPU，资历显然比不过英伟达和AMD。所以在日常生活中，英特尔显卡并不常见。

这两家各有优势：

- **NVIDIA（N卡）**：市场份额大，驱动稳定，AI和深度学习生态好（CUDA技术），光线追踪技术成熟，支持DLSS（深度学习超采样）。如果你是AI用户或者想畅玩光追游戏，N卡会是你不错的选择。
- **AMD（A卡）**：性价比高，同价位下显存通常更大，光栅化性能强（传统游戏性能），最新的FSR技术也在追赶DLSS。预算有限又想体验高性能的可以考虑。
### 英伟达

我们先来看几个例子：RTX 4060、RTX 4070 Ti、RTX 4090 D。

首先是前缀GTX和RTX[^3]。这个指是英伟达的产品线。GTX是英伟达旧款显卡产品线，现已弃用。RTX是英伟达旧款显卡产品线，支持光线追踪。本质上，这个信息并不是很重要。

接下来的后两个数字就是这个显卡的代数，比如RTX 4060就是第40代系的产品。与CPU一样，只有相同代系的显卡，后面的数字才有对比价值，如果代系不一样的显卡进行对比，就需要看各种测试结果了。

然后就是再后两位数字了，这代表了这张显卡在同一代系的定位，一般遵循以下原则：

- 50：入门
- 60：主流
- 70：高端
- 80：旗舰
- 90：顶级

最后就是后缀字母了，可以参考下表：

| 英伟达显卡常见后缀  | 含义                                                   |
| ---------- | ---------------------------------------------------- |
| Ti         | 比纯数字的性能更强，如：RTX 4060 < RTX 4060Ti                    |
| Super（简称S） | 介于纯数字和Ti的中间值，如：RTX 3060 < RTX 3060Super < RTX 3060Ti |
| D          | 为符合美国出口标准而为中国市场提供的定制版，性能通常比纯数字版还要弱。                  |
| MX         | 亮机卡，大型游戏基本带不动                                        |

### AMD

AMD的显卡命名规则略有不同。

首先，AMD的显卡通称RX（Radeon，中文名叫镭龙），这就不用管了。

接下来，AMD显卡的第一位数字就是显卡代数了，如RX 7600，这张显卡就是第七代镭龙显卡了。

然后，后面三位就是这张显卡的市场定位了，一般来讲遵守以下规则：

- 900 级：旗舰
- 800 级：高端
- 700 级：中高端
- 600 级：中端
- 500 及以下：入门 / 办公

最后就是后缀了，常见的AMD显卡后缀可以参考下表：

| AMD显卡常见的后缀 | 含义                                      |
| ---------- | --------------------------------------- |
| XTX        | 同代显卡的最强旗舰卡，性能在同代中最强                     |
| XT         | 介于纯数字和XTX的一张显卡，性能介于XTX显卡与纯数字显卡的中间       |
| GRE        | 为符合美国出口标准而为中国市场提供的定制版，性能介于纯数字显卡和XT显卡的中间 |

# 内存（Memory）

它应该长这样👇：

![[内存条.png]]

内存，也叫内存条，是计算机中的临时存储设备。它的工作是临时存储CPU待处理数据。需要注意的是，大部分软件为了提升效率，一般都会把修改的内容先存在内存里面，当你下达保存指令时（按下“保存”按钮或狂按Ctrl+S）,软件才会把修改的内容写入硬盘，而内存里面的数据在断电关机以后会全部消失，所以这样就解释了为什么电脑在蓝屏死机以后，你写的Word文档或者你使用PS修图以后的内容会全部丢失，功亏一篑。

> 请参阅[[前述]]，煮包在这里详细的介绍了这两者的区别。

## DDR4?DDR5?

目前市面上主流的家用计算机内存分为DDR4和DDR5两代，如下图所示：

- **DDR4**：老一代，价格便宜，时序较低（延迟较小）。对于追求性价比的用户来说，DDR4平台（如搭配12代/13代酷睿或AMD AM4平台）仍然是不错的选择。
- **DDR5**：新一代，频率更高、带宽更大，价格稍贵。如果是新装电脑，煮包建议直接上DDR5，战未来。

> [!WARNING] 注意
> DDR4和DDR5的插槽物理结构不一样，不能混插！买主板的时候要看清楚支持的是DDR4还是DDR5。

## 内存疯狂涨价，普通消费者该怎么办？（2026年5月16日）

好吧，其实这个时候说这个事件多少有一点炒冷饭的意思，毕竟内存价格已经开始在降价了。

> [!quote] 发生了什么？
> 2025年第一季度，由于AI产业对内存需求大幅度上涨，再加之多家内存制造厂家主动减低产能，导致消费级的内存大幅涨价。厂家们把这个事情定位成产能与需求失衡，不过大多数消费者把这个事件定位成“饥渴营销”

如今内存价格一路疯涨，行情居高不下，普通消费者无需盲目跟风升级硬件。非刚需尽量暂缓装机与内存扩容计划，继续沿用现有设备凑合用，通过清理后台、精简程序释放运行空间。若确实急需添置，优先选择性价比更高的DDR4内存，也可入手成色靠谱的正规二手内存，按需选购够用容量即可，切勿大额囤货被套牢，耐心等待产能回暖、价格回落再做更换打算。
# 硬盘（Hard Disk）

它应该长这样👇：

![[硬盘.png]]

硬盘是计算机的永久存储设备，用来存放操作系统、软件、游戏、文档、照片等所有数据。跟内存不同，硬盘中的数据在关机后不会丢失。

目前硬盘主要分为两种：固态硬盘（SSD）和机械硬盘。接下来我们来详细的介绍一下这两种硬盘。
## 固态硬盘（SSD，Solid State Drive）

SSD是目前电脑的主流选择，速度快、不怕震动、噪音小、功耗低。现在的笔记本电脑和台式机基本都标配SSD了。

SSD又分为几种接口，这里煮包只介绍最常见的两种：

- **M.2 NVMe SSD**：像一根小口香糖，直接插在主板上。速度极快（读取可达每秒几千MB），是目前装机首选。
- **SATA SSD**：跟2.5英寸的薄盘一样大小，通过SATA线连接主板。速度比NVMe慢得多（上限约550MB/s），但比机械硬盘快很多。老电脑升级可以考虑。

所以，总结一下就是：新电脑直接上M.2 NVMe SSD，容量大约1TB。如果是老电脑升级，主要是看电脑主板支不支持M.2 NVMe协议。

## 机械硬盘（HDD，Hard Disk Drive）

HDD是传统的硬盘，内部有磁盘和磁头，靠旋转磁盘来读写数据。它的优点是容量大、价格便宜，缺点是速度慢、怕摔、有噪音。

目前HDD主要用作数据仓库（冷存储），比如存放不常用的文件、电影、备份等。常见容量从1TB到100TB不等。
# 电源（Power）

它应该长这样👇：

![[电源.jpg]]
最后一个就是电源啦，电源（Power Supply Unit，PSU），负责把家里的220V交流电转化为计算机各部件需要的直流电。别看它不起眼，电源的质量直接决定了一台电脑能否稳定运行。
## 电源的转换效率与金牌认证

电源将交流电转为直流电时会损耗一部分能量，转换效率越高越省电。80 PLUS认证就是衡量电源转换效率的标准：

| 等级 | 转换效率（50%负载时） | 建议 |
| --- | --- | --- |
| 白牌 | 80% | ❌ 不太推荐 |
| 铜牌 | 82% | ❌ 预算实在紧张再考虑 |
| 银牌 | 85% | ⚠️ 一般不常见 |
| 金牌 | 87% | ✅ **主流选择，推荐** |
| 铂金 | 89% | ✅ 高配主机推荐 |
| 钛金 | 90% | 💎 顶级配置不差钱可选 |

煮包个人建议：**金牌电源是甜点选择**，性价比最高。
