# 总述

计算机类型一般可以分为以下几种：台式电脑、笔记本电脑和一体机电脑。接下来，我将列出一份表格，详细讲述不同的电脑类型。

|             | 优点                         | 缺点                            |
| ----------- | -------------------------- | ----------------------------- |
| 台式电脑（主机+屏幕） | 性能强劲；维护与升级容易且便宜。           | 断电即关机，断电前的工作成果直接拜拜；移动性差；售后困难。 |
| 笔记本电脑       | 便携性强；品牌机售后有保障；电池可以应对短时间工作。 | 官方配件较贵；维护与升级困难。               |
| 一体机电脑       | 颜值高；噪音较小。                  | 既牺牲了便捷性，还牺牲了性能，属于是集合了以上所有缺点。  |


> [!INFO] 省流
> 想要性能购买台式电脑，想要便携购买笔记本电脑。煮包不推荐购买一体机电脑。

# 台式电脑

它应该长这样👇：

![[台式电脑.png]]

一台标准的多媒体台式电脑包含以下几个部分：

- 主机（既上图中的黑色箱子）
- 显示器（主机右侧的黑色长方形）
- 音响（主机和显示器旁边的木质纹理的设备）
- 键盘（屏幕正前方的长方形设备）
- 鼠标（键盘右侧的设备）

## 主机

主机是整个计算机的核心。以上的所有设备都要接到主机上才能正常使用。可以说，主机是一台电脑的“电子大脑”。

大部分2000~2010左右的主机配备有CD/DVD读取刻录器，现在大多已经被USB接口取代。

主机里的配件还可以再分，由于篇幅限制，煮包将这一部分放在了[[认识计算机硬件]]详细讲述。

## 显示器（输出设备）

显示器是主机的输出设备之一，负责将主机渲染出来的画面投影出来供操作者观看。一般来讲，市面上的显示器大小不同，但一般来讲都是兼容主机的。

## 音响（输出设备）

音响是主机的另一个输出设备，负责将主机输出的声音播放出来供操作者聆听。同样的，音响的形状、大小、样式各不相同，但基本与计算机兼容。

## 键盘与鼠标（输入设备）

这个大家就很清楚了。这两个设备都作为计算机的输入设备。其中，鼠标负责控制系统的指针进行交互，键盘负责将文字输入到计算机。

鼠标的用法煮包就不多介绍了。关于键盘快捷键的介绍煮包放在了[[常见的键盘快捷键]]这一章节里介绍。
