
# 提示块类型更改

提示块的语法如下：

```
> [!<指定图标>] 标题
> 内容
```

如：

> [!TIP] 技巧
> 记得多加练习。

常用图标：

| 类型标识       | 含义       | 适用场景             |
| ---------- | -------- | ---------------- |
| `TIP`      | 提示 / 小技巧 | 实用方法、操作技巧        |
| `INFO`     | 信息 / 说明  | 补充信息、背景介绍        |
| `WARNING`  | 警告       | 需要注意的风险、易错点      |
| `CAUTION`  | 小心 / 警惕  | 可能导致问题的操作        |
| `DANGER`   | 危险       | 高危操作（如数据删除、权限修改） |
| `QUESTION` | 问题       | 待解答的疑问、思考点       |
| `SUCCESS`  | 成功 / 完成  | 操作成功的提示、结果确认     |
| `FAILURE`  | 失败 / 错误  | 操作失败的原因、错误说明     |
| `EXAMPLE`  | 示例       | 代码示例、场景示例        |
